hexo博客的搭建及配置

发表于 2020-09-08 更新于 2020-09-10 分类于 hexo 阅读次数：

一、安装必备软件

  • 安装node

    hexo是基于Node.js框架的静态博客程序

  • 安装git

    用于本地文件与服务器文件的同步

  • 安装一款markdown编辑器

    推荐Typora或者用Visual Studio Code编辑器（安装markdown插件）

  • 熟悉npm命令用法

    全局安装hexo基础框架

  • Github账号及账号同名的库

    用来托管你的静态博客内容；可通过如下步骤操作

    • 创建一个账号和创建一个库

      假如你的账号为tinykeen,则库名为tinykeen.github.io

    • 本地生成SSH密钥

      默认密钥对（公钥，私钥）生成再~/.ssh目录下面，

  • 配置公钥

    拷贝公钥的内容，添加、配置到Github网站的对应地方

  • 密钥验证

  • 用来初始化博客网站的信息
  • 配置博客网站的主题及其他配置项

四、部署到Github

  • 修改配置文件

    根据需要，修改符合自己博客主题的配置项，如博客名称等，发布部署的配置项是

    deploy:
    type: git —-发布方式
    repository: https://github.com/tinykeen/tinykeen.github.io.git —对应第一章节中创建的账号同名库
    branch: master

  • 发布博客

发布如果提示有问题，请安装发布插件npm install hexo-deployer-git –save
